[明道云](https://www.mingdao.com) 是一个企业软件的快速设计和开发工具。但不需要任何代码编写，普通业务人员就能掌握使用。通过灵活的功能组件，让企业可以搭建个性化的 CRM、ERP、OA、项目管理、进销存等系统，你可以用它管理生产、销售、采购、人事等所有企业活动。明道云私有部署基于镜像模式，旨在提供给用户一个能快速安装与体验的 APaaS 平台。
